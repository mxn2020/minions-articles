---
title: Installation
description: Install Minions Articles packages
---

## TypeScript / Node.js

```bash
npm install @minions-articles/sdk minions-sdk
```

## Python

```bash
pip install minions-articles
```

## CLI

```bash
npm install -g @minions-articles/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
