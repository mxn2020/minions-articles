# Minions Articles

Article drafts, revisions, SEO metadata, and publish status

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-articles/sdk`) — Core library for Node.js
- **Python SDK** (`minions-articles`) — Python bindings
- **CLI Tool** (`@minions-articles/cli`) — Command-line interface
- **Web App** — Interactive playground at [articles.minions.wtf](https://articles.minions.wtf)
